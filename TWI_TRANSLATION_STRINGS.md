# AsantePost - UI Strings for Twi Translation

**Instructions for Translator:**
- Fill in the **Twi** column for each English string
- Keep placeholders like `{name}`, `{count}`, `${amount}` unchanged - they are replaced by the app with actual values
- Cultural/Asante terms (Akom, Anansesem, Akwasidae, Adinkra, etc.) should remain as-is since they are already Twi
- Some strings may need culturally appropriate equivalents rather than direct translations
- Mark any string you are unsure about with `[REVIEW]` so the developer can discuss
- If a string doesn't make sense to translate (e.g., "PayPal", "MTN Mobile Money"), write `[KEEP ENGLISH]`
